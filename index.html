Memory Game

How to Play

Welcome to my Memory Game! Currently you can mess around with the buttons below. If you would like to start the game select a gamemode. Repeat the pattern by clicking the buttons in the correct sequence.

Gamemodes
Standard: Random Pattern, limited to 8 rounds, get to the end to win.
Endless: See how far you can get with an endless cycle of randomized patterns.
Speed Up: Speeds up the game after every round. Endless Amount of Rounds with a random pattern every time.
3 Strikes: Three Strikes and you're out, after each mistake you will be alerted. Endless Amount of Rounds with a random pattern every time.

WARNING: If stop button is clicked while sequence is in the middle of an audio cue it may glitch. Please wait a moment in between switching gamemodes.
